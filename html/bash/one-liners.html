Histogram in bash¶

Create a nice looking histogram based on the number of elements found when counting with uniq.

$ cat somefile | sort | uniq -c | perl -lane 'print $F[1], "\t", $F[0], "\t", "=" x ($F[0] / 3 + 1)'

Python: Virtual envs

This page shows an ideal python workflow using virtual environments, for both python 2 and python 3.

Python 2

virtualenv -p path/to/python2 path/to/venv

Python 3

python -m venv path/to/venv

Generic

Activate the environment:

source path/to/venv/bin/activate

Deactivate the environment:

deactivate
